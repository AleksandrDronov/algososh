# Визуализатор алгоритмов

### Используемые технологии:
1. ReactJS
2. TypeScript
3. Jest
4. Cypress

### Запуск проекта:

1.Клонировать репозиторий
  git clone git@github.com:AleksandrDronov/algososh.git

2.Войти в папку проекта
  cd algososh

3.Установить зависимости
  npm install

4.Запуск
  npm start
